## ⚖️ 검증 결과 — "기술적 근거의 예측력 ≈ 동전던지기" (2026-06)

`factor_backtest.py`(근거별 IC) + `factor_learn.py`(로지스틱 다변량 학습)로 코스피·코스닥
대형주 50종목·약 9만 종목-일을 시간순 검증한 결론:

- **기술적·가격 근거만으로는 미래 방향을 사실상 못 맞춘다.** 로지스틱 회귀 검증 AUC가
  5일 0.504 / 20일 0.520 / 60일 0.443 — 어느 기간도 베이스라인(항상 다수)을 못 넘었다.
  (RF/XGBoost 앙상블도 정확도 ≈ 0.50.) 코드 문제가 아니라 **효율적 시장의 본질**.
- 따라서 **점수 가중치를 '학습 가중치'로 교체해도 의미 없다**(학습할 신호 자체가 없음).
  단변량 IC에서 "추세가 역효과"처럼 보이는 것도 모델이 무작위 수준이라 신뢰 불가.
- **결론: 점수제는 '예측기'가 아니라 '투명한 현황 체크리스트'.** 기대수익의 원천은
  ① 재료(공시·뉴스 catalyst), ② 밸류에이션·재무(장기), ③ 리스크 관리, ④ 투명성.
  → 단기 방향 예측에 과신 금지. (재검증: `python factor_backtest.py`, `python factor_learn.py`)

## 🔭 더 해볼 만한 것 (검증 결과 기반 — "진짜 신호가 있을 법한 곳")

**A. 예측력이 있을 가능성 (학술적 근거)**
- [ ] **재료(공시) 이벤트 스터디** — DART 공시일은 과거 데이터가 있어 **백테스트 가능**.
      "호재 공시 후 N일 드리프트"가 실제로 있는지 측정 → catalyst 가중치 강화 근거. (거의 유일하게 검증 가능한 펀더멘털 축)
- [ ] **횡단면 상대강도** — "내일 오를까(시계열)" 대신 "같은 날 종목 중 상대적으로 강한가". 시계열보다 견고.
- [ ] **소형주(비효율 시장) 예측력** — `factor_learn.py`를 소형·중형주로 재실행 (대형주는 효율적이라 ≈0.5).
- [ ] **장기 가치/퀄리티 프리미엄** — 저PER·저PBR·고ROE 바스켓 장기 보유 백테스트. (무료 과거 펀더멘털 부재가 걸림돌 → 적립식 수집)

**B. 예측이 빗나가도 돈 되는 축**
- [ ] **리스크 관리·포지션 사이징** — 변동성 기반 손절폭·비중 자동 산출(켈리/변동성 타겟팅).
- [ ] **변동성 예측(GARCH 등)** — 변동성은 군집성이 있어 예측 가능 → 손절/목표 폭 정밀화.

**C. 검증·운영 인프라**
- [ ] **유니버스·기간 확대 + 생존편향 제거** (상장폐지 포함).
- [ ] **신호 적중 추적(라이브)** — 라이브 신호 vs 실제 결과 누적 → 메타분석.
- [ ] **이벤트 스터디 도구화** — "공시/급등 후 수익 분포" 스크립트.
