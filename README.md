# 📈 StockView — 나만의 주식 투자 판단 도우미

한국 주식(KOSPI/KOSDAQ)과 미국 주식(NASDAQ/NYSE)을 머신러닝과 기술적 지표로 분석해,
**"지금 살까, 아니면 뺄까"** 를 판단하도록 돕는 개인용 웹 앱입니다.
종목을 검색하면 가격 차트, 다음 거래일 상승/하락 ML 예측, 그리고 여러 신호를 종합한
**매수 · 관망 · 매도 판단**과 그 근거를 보여줍니다.

> ⚠️ 주가는 100% 맞출 수 없습니다(특히 단기). 이 도구의 신호는 ML 예측과 기술적
> 지표(추세·RSI·모멘텀)를 종합한 **참고용 판단**이며 틀릴 수 있습니다. 최종 결정과
> 책임은 본인에게 있습니다. 그래서 과신을 막기 위해, 신호와 함께 **이 모델이 과거에
> 실제로 얼마나 맞췄는지(정확도·베이스라인 대비 우위)** 도 항상 같이 보여줍니다.

## 구조

```
StockView/
├── backend/          # Python + FastAPI
│   ├── data.py       # FinanceDataReader로 한국+미국 주식 데이터 수집
│   ├── features.py   # 기술적 지표 → ML 피처 (이동평균, RSI, 변동성 등)
│   ├── model.py      # RandomForest 학습 / 예측 / 시간순 백테스트
│   └── main.py       # API 엔드포인트
└── frontend/         # React (Vite) + Recharts
    └── src/
        ├── App.jsx
        └── components/  # SearchBar, PriceChart, PredictionPanel
```

## 실행 방법

### 가장 간단 — `make` (권장)

```bash
make install   # 최초 1회: 백엔드 venv + 프론트 node_modules 설치
make           # 백엔드(:8000) + 프론트(:5173) 동시 실행, Ctrl-C로 둘 다 종료
```

브라우저에서 **http://localhost:5173** 접속.

| 명령 | 설명 |
|------|------|
| `make` | 백엔드 + 프론트 **동시 실행** |
| `make backend`  | 백엔드만 (코드 변경 시 자동 리로드) |
| `make frontend` | 프론트만 |
| `make install`  | 의존성 설치 |
| `make clean`    | venv / node_modules 삭제 |

> Windows에 `make`가 없다면: `winget install ezwinports.make` 설치 후 **터미널을 새로 열면** `make`가 잡힙니다.
> Makefile은 셸을 Git Bash로 고정하므로 Git이 설치돼 있어야 합니다.

### 수동 실행 (make 없이, 터미널 2개)

```powershell
# 터미널 1 — 백엔드
cd backend
./venv/Scripts/python.exe -m uvicorn main:app --port 8000 --host 127.0.0.1

# 터미널 2 — 프론트엔드
cd frontend
npm run dev
```

(최초 설치: `cd backend && python -m venv venv && ./venv/Scripts/python.exe -m pip install -r requirements.txt`,
그리고 `cd frontend && npm install`)

## API

| 엔드포인트 | 설명 |
|-----------|------|
| `GET /api/search?q=삼성` | 종목명/코드 검색 (한국+미국, 예: `삼성`, `AAPL`) |
| `GET /api/stock/{code}`  | 일봉 OHLCV 시세 (차트용) |
| `GET /api/predict/{code}`| ML 예측 + 백테스트 평가 지표 |
| `GET /api/preview/{code}`| 목록용 빠른 예측 (방향+확률만) |
| `GET /api/news/{code}`   | 최근 뉴스 헤드라인 (한국=네이버, 미국=구글 뉴스) |
| `GET /api/lists`         | 추천 목록 카탈로그 |
| `GET /api/list/{id}`     | 목록별 종목 (예: `krx_cap100`, `us_cap100`, `krx_gainers`) |

## 투자 판단 기준 (매수 / 관망 / 매도)

여러 신호에 **점수를 매겨 합산**하고, 합계 점수로 행동을 정합니다. (모든 근거는 화면에 그대로 표시)

| # | 기준 | 점수 | 데이터 출처 |
|---|------|------|------------|
| 1 | **ML 예측** (다음날 상승확률) | ≥55% +2 / ≥52% +1 / ≤48% −1 / ≤45% −2 | RandomForest (아래 참고) |
| 2 | **추세** (20·60일 이동평균) | 둘 다 위 +1.5 / 둘 다 아래 −1.5 | 시세 |
| 3 | **RSI** | ≤30 과매도 +1 / ≥70 과매수 −1 | 시세 |
| 4 | **모멘텀** (최근 5일) | +3%↑ +0.5 / −3%↓ −0.5 | 시세 |
| 5 | **밸류에이션** | PER<10 +0.5 / PER>40 −0.5 / PBR<1 +0.5 / PBR>5 −0.3 | 네이버(KR)·stockanalysis(US) |
| 6 | **수급** (외국인·기관 순매수) | 동반 순매수 +1 / 동반 순매도 −1 | 네이버 (KR만) |
| 7 | **뉴스 감성** | 긍정 우세 +1 / 부정 우세 −1 (헤드라인 사전 분석) | 네이버·구글 뉴스 |
| 8 | **재료(이벤트)** | 자사주매입·증설·최초기술·대형수주 등 +0.5씩(최대 +1) / 악재 −1 | 뉴스 키워드 |
| 9 | **애널리스트 의견** | Buy +0.5 / Sell −0.5 | stockanalysis (US만) |

**합계 → 행동**: `≥3` 매수 우위 · `≥1` 약한 매수 · `−1~1` 관망 · `≤−1` 약한 매도 · `≤−3` 매도 우위

**신뢰도 보정**: 그 종목에서 ML 모델의 백테스트 `edge`(베이스라인 대비 우위)가 0 이하이면
신뢰도를 낮추고 경고를 표시 → 과신 방지.

**손절·목표가**: 변동성(ATR 14일) 기반. 손절 = 현재가 − 1.5×ATR, 목표 = 현재가 + 2×ATR (손익비 ≈ 1.3:1).

### ML 예측 모델

1. 과거 일봉에서 피처 생성 — 단기/중기 수익률, 이동평균 대비 위치, 변동성, 거래량 변화, RSI, 최근 가격대 내 위치.
2. "다음날 종가 > 오늘?"을 타깃으로 RandomForest 학습.
3. **시간순** 앞 80% 학습 / 뒤 20% 검증 (시계열이라 셔플 금지).
4. 검증 정확도를 베이스라인과 비교해 `edge` 계산 → edge>0 이어야 모델이 의미 있음.

### 한계와 다음 단계

- **재료 감지(8번)는 뉴스 제목 키워드 기반의 거친 추정**입니다. "자사주 매입·증설·최초 기술·좋은 공시"를
  정확히 잡으려면 **DART 전자공시 API**(무료, 키 발급 필요) 연동이 필요합니다.
- **미국 수급**(기관 순매수)은 무료 일별 데이터가 없어 미반영입니다.
- 단기 주가 예측은 본질적으로 한계가 있습니다. 이 신호는 **참고용**이며 최종 판단은 본인 몫입니다.

### 6. 판단 근거·정확도 더 높이기 (다음 후보)
> 지금까지 근거는 많이 늘렸으니, 이제 **각 근거의 신뢰도와 모델 정확도**를 끌어올리는 단계.

**모델 정확도**
- [ ] **ML 확률 보정(calibration)** — RandomForest 확률을 isotonic/sigmoid로 보정해
      "상승확률 55%"가 실제 55%에 가깝게. 신호 신뢰도·임계값 정확도↑ (체감 효과 큼).
- [ ] **앙상블 예측을 신호에 반영** — 지금 신호는 RF 확률만 사용. RF·XGBoost(·LSTM) 평균/투표로
      단일 모델 편향↓. (모델 비교는 이미 표시 중 → 그 출력을 신호에 연결만 하면 됨)
- [ ] **하이퍼파라미터 튜닝 + 피처 중요도 점검** — 과적합 줄이고 효과 없는 피처 정리.
- [ ] **타깃 재정의** — '다음날 상승'(이진) 대신 '+0.5% 이상 의미있는 상승' 등으로 노이즈↓.

**근거 신뢰도·추가 지표**
- [ ] **재무 추세** — ROE·영업이익률이 개선 중인지 악화 중인지(분기 흐름), 어닝 서프라이즈(컨센서스 대비).
- [ ] **수급 추세 정교화** — 외국인·기관 '연속 순매수 일수'·금액 추세 (현재는 5일 순매수 여부만).
- [ ] **실적 발표일(어닝) 임박 경고** — 발표 D-day 표시 + 발표 전후 변동성 주의.
- [ ] **진짜 동종업계(peer) 상대강도** — 현재는 시장지수 대비. 같은 업종 종목들 평균 대비로 정교화.
- [ ] **KR 공매도·섹터명 보강** — KRX 데이터로 한국 공매도 비중·업종 분류 채우기(현재 US만).
- [ ] **배당 매력** — 배당수익률·연속 배당성장을 근거로 (안정·인컴 투자 관점).
- [ ] **뉴스 감성 LLM화** — 키워드 사전 → Gemini가 헤드라인 감성을 0~1로 채점(정확도↑, 무료).

**활용·운영**
- [ ] **edge 높은 종목만 강조** — 백테스트에서 모델이 베이스라인을 이긴 종목만 추천/신뢰 표시.
- [ ] **AI 분석 결과 캐싱** — 같은 날 같은 종목은 재사용해 속도·호출량↓.
- [ ] **신호 적중 추적** — 과거 신호 vs 실제 결과를 기록해 어떤 근거가 잘 맞는지 가중치 학습.
